Selectors in jQuery

5 Feb 2008 edit

Just a note to self: Quick Tip - Optimizing DOM Traversal. Avoid bare classname selectors like $('.foo') when possible; qualify them with an element type or, ideally, the ID of a parent element: $('div.foo') or $('#bar .foo').
